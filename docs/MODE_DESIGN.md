# Quant-ABC 运行模式设计

## 概述

定义 quant-abc 系统的三种运行模式。

## 模式定义

| 模式 | 用途 | 采样方式 | 数量 |
|------|------|----------|------|
| **test_mode** | 冒烟测试 | 随机采样 | 50只 |
| **dev_mode** | 开发验证 | SH/SZ分层采样 | 沪市25 + 深市25 |
| **production_mode** | 正式推荐 | 全量候选池（过滤后） | 5289只 |

## 配置

```python
# config.py
RUN_MODE = "production"  # test / dev / production
```

## 生产模式股票池 (production_mode)

### 股票池统计

| 市场 | 代码前缀 | 数量 |
|------|---------|------|
| 沪市主板 | .SH | 1648 |
| 深市主板 | .SZ | 1413 |
| 创业板 | 30xxx.SZ | 1349 |
| 科创板 | 688xxx.SH | 593 |
| 北交所 | .BJ | 286 |
| **总计** | | **5289** |

### 过滤规则

| 条件 | 说明 |
|------|------|
| ST股票 | 排除名称含 ST/*ST/S*ST 的股票 |
| 新股 | 排除上市不足60个交易日的股票 |
| 停牌 | 排除收盘价为0的股票 |
| 退市 | 排除已退市股票 |

### 市场说明

- **全A股**: 覆盖沪市、深市、创业板、科创板、北交所
- **不含**: 港股、美股等境外市场

## 各模块使用

| 模块 | 模式 | 说明 |
|------|------|------|
| 测试脚本 | test/dev | 快速验证 |
| 盘前推荐 | production | 正式推荐 |
| 回测系统 | production | 完整回测 |

## 为什么不能随机采样

production_mode 必须使用全量数据，因为：
1. 随机采样不具代表性
2. 回测结果不可靠
3. 实际交易无法覆盖全市场
