# Quant-ABC 运行模式设计

## 概述

定义 quant-abc 系统的三层股票池架构。

## 三层股票池定义

| 池子 | 定义 | 数据来源 | 更新频率 |
|------|------|----------|----------|
| **全市场股票池** | 全部A股 | get_stock_basic() | 日更 |
| **静态过滤池** | 全市场 - ST - 新股 | 静态字段判断 | 日更 |
| **动态过滤池** | 静态池 - 停牌 - 低成交 | 实时行情 | 盘前 |

## 三层数量（示例）

| 池子 | 数量 | 过滤规则 |
|------|------|----------|
| 全市场 | 5488 | - |
| 静态过滤 | 5290 | -ST -新股(60天) |
| 动态过滤 | 待获取 | -停牌 -低成交 |

## 两层过滤定义

### 静态过滤（Static Filter）

**位置**: stock_selector.py - get_candidate_pool()

**规则**:
| 规则 | 字段 | 条件 |
|------|------|------|
| 排除ST | name | 含 ST/*ST/S*ST |
| 排除新股 | list_date | 上市<60交易日 |

**状态**: ✅ 已实现

### 动态过滤（Dynamic Filter）

**位置**: data_fetcher.py - 获取行情后执行

**规则**:
| 规则 | 字段 | 条件 |
|------|------|------|
| 排除停牌 | close | = 0 |
| 排除低成交 | amount | < 100万元/日 |

**状态**: ⚠️ 需要行情数据支持

## 模式定义

| 模式 | 用途 | 采样方式 |
|------|------|----------|
| test_mode | 冒烟测试 | 随机采样 50只 |
| dev_mode | 开发验证 | SH/SZ 分层采样 |
| production_mode | 正式推荐 | 全量三层过滤 |

## 配置

```python
# config.py
RUN_MODE = "production"
```

## 缓存策略

| 数据类型 | 缓存时间 | 更新策略 |
|----------|----------|----------|
| 股票列表 | 1天 | 每日盘前 |
| 静态过滤池 | 1天 | 每日盘前 |
| 实时行情 | 4小时 | 盘前增量 |
